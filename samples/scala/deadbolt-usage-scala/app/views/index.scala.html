Set-up

The current user has the roles "foo" and "bar". For each example, the required roles will be stated, along with the expected result.

Controller restrictions

SBRestrict

  • Restrict One
    • restriction requiring foo
    • should be visible
  • Restrict Two
    • restriction requiring foo and bar
    • should be visible
  • Restrict Three
    • restriction requiring foo but not bar
    • should NOT be visible
  • Restrict Four
    • restriction requiring hurdy
    • should NOT be visible
@*

SBRestrictions

  • Restrictions One
    • restriction requiring foo and bar
    • should be visible
  • Restrictions Two
    • restriction requiring foo or bar
    • should be visible
  • Restrictions Three
    • restriction requiring hurdy AND gurdy, or foo
    • should be visible
  • Restrictions Four
    • restriction requiring foo OR NOT bar
    • should be visible
  • Restrictions Five
    • restriction requiring hurdy and NOT foo
    • should NOT be visible
*@

SBDynamic

  • Pure luck
    • restriction requiring the system time in milliseconds to be exactly divisible by 2
    • may be visible, depends on your luck
  • No way, Jose
    • restriction using the same resource name as "pure luck" above, but specifying a different handler
    • should NOT be visible

SBRoleHolderPresent

  • Logged in
    • restriction requiring a role holder to be present (e.g. someone is logged in)
    • a user is logged in, so this is accessible
  • Not logged in
    • restriction requiring a role holder to be present (e.g. someone is logged in)
    • a user is not logged in, so this is NOT accessible

SBRoleHolderNotPresent

  • Logged in
    • restriction requiring a role holder to be present (e.g. someone is logged in)
    • a user is not logged in, so this is NOT accessible
  • Not logged in
    • restriction requiring a role holder to be present (e.g. someone is logged in)
    • a user is not logged in, so this is accessible

Tag restrictions

restrict

Requires: foo and bar. Expected result: contents are visible
@restrict(handler, List(as("foo", "bar"))) { This content should be visible }
Requires: foo, bar, and hurdy. Expected result: contents are NOT visible
@restrict(handler, List(as("foo", "bar", "hurdy"))) { This content should NOT be visible }

restrictOr

Requires: foo and bar. Expected result: the original contents are displayed
@restrictOr(handler, List(as("foo", "bar"))) { This content should be visible }{This will not be visible}
Requires: foo, bar, and hurdy. Expected result: Whoops
@restrictOr(handler, List(as("foo", "bar", "hurdy"))) { This content should NOT be visible }{Whoops}

roleHolderPresent

Requires: a role holder to be present. Expected result: A role holder is present, so the contents are visible
@roleHolderPresent(handler) { This content should be visible }
Requires: a role holder to be present. Expected result: A role holder is NOT present, so the contents are NOT visible
@roleHolderPresent(new security.MyUserlessDeadboltHandler) { This content should NOT be visible }

roleHolderPresentOr

Requires: a role holder to be present. Expected result: A role holder is present, so the original contents are displayed
@roleHolderPresentOr(handler) { This content should be visible }{Whoops}
Requires: a role holder to be present. Expected result: A role holder is NOT present, so...whoops
@roleHolderPresentOr(new security.MyUserlessDeadboltHandler) { This content should NOT be visible }{Whoops}

roleHolderNotPresent

Requires: a role holder to NOT be present. Expected result: A role holder is present, so the contents are NOT visible

Requires: a role holder to NOT be present. Expected result: A role holder is NOT present, so the contents are visible
@roleHolderNotPresent(new security.MyUserlessDeadboltHandler) { This content should be visible }

roleHolderNotPresentOr

Requires: a role holder to NOT be present. Expected result: A role holder is present, so...whoops

Requires: a role holder to NOT be present. Expected result: A role holder is NOT present, so the original contents are displayed
@roleHolderNotPresentOr(new security.MyUserlessDeadboltHandler) { This content should be visible }{Whoops}
