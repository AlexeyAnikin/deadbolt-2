RoleHolderPresent

This is one of the simplest constraints in Deadbolt. It simply requires that a user be present (i.e. logged in).

Specification
Result
RoleHolderPresent(new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
RoleHolderPresent(new MyUserlessDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
