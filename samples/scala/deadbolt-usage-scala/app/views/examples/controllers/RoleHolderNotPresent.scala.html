RoleHolderNotPresent

The counterpart to RoleHolderPresent. This constraint requires that a user isn't present (i.e. no-one is logged in).

Specification
Result
RoleHolderNotPresent(new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
RoleHolderNotPresent(new MyUserlessDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
